◆ ADC_CFGR_EXTSEL

#define ADC_CFGR_EXTSEL (   __HANDLE__,
  __EXT_TRIG_CONV__ 
)    (__EXT_TRIG_CONV__)

For devices with 3 ADCs: Defines the external trigger source for regular group according to ADC into common group ADC1&ADC2 or ADC3 (some triggers with same source have different value to be programmed into ADC EXTSEL bits of CR2 register). For devices with 2 ADCs or less: this macro makes no change.

Parameters
<strong>HANDLE</strong>ADC handle
<strong>EXT_TRIG_CONV</strong>External trigger selected for regular group.
Return values
Externaltrigger to be programmed into EXTSEL bits of CR2 register
